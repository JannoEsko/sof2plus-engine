# SoF2Plus Engine

This repository was initially made to hold the necessary changes for 1fxplus. 
Nowadays the master branch is not being kept up to date. Work is being done in other branches.

For SoF2 1.00 (protocol 2002) engine, please take a look at [sof2plus-1.00 branch](https://github.com/JannoEsko/sof2plus-engine/tree/sof2plus-1.00).
For SoF2 Multiprotocol (serving clients from both SoF2 1.00, protocol 2002 and SoF2 1.03, protocol 2004), please take a look at [sof2plus-multiprotocol branch](https://github.com/JannoEsko/sof2plus-engine/tree/sof2plus-multiprotocol).
